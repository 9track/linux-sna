Quick installation of your Linux-SNA server (you dont even need to read about SNA) but
dont be blindows minded, please! :P


1. Get all the files shown in www.linux-sna.org.

2. Please remember:

  • Linux kernel (same version as your Kernel Patch) from www.linuxkernel.org
  • Kernel Patch: net-fun-0.0XX-2.X.XX.diff.gz where 2.X.XX is the kernel
    version it uses.-Library (really important files) libsna-0.00X.tar.gz
  • sna-tools-0.0XX.tar.gz A collection of programs that form the base set of
    the Linux-SNA networking distribution for the Linux operating system.
  • asuite-0.0X.tar.gz is a suite containing the tools aping, aftp and some
    other a- please refer (at least) to http://pclt.cis.yale.edu/pclt/COMM/SNA.HTM.
  • etc-sna-0.00X.tar.gz this will install the Linux-SNA environment on your system

3. How to set up all this.

  • tar -zxvf linux-2.X.XX.tar.gz
  • gzip -d net-fun-0.0XX-2.X.XX.diff.gz-patch -p0 < net-fun.0.0XX-2.X.XX.diff
  • now go to usr/src/linux and type make menuconfig
  • configure your kernel activating this options:
    • ANSI/IEEE 802.2 Data link layer protocol
    • ANSI/IEEE 802.[3,4,5] Physical layer protocols
    • SNAP protocol layer
    • Advanced Linux Network Buffers
    • Common Programming Interface Communications (CPI-C)
    • Advanced Program-to-Program Communications (APPC)
    • SNA Networking
    • SNA: IEEE 802.2 device support
    • SNA: LAN Address Resolutions
    • SNA: Loopback device support
  • the rest of your kernel settings you are supposed to know. If not,Kernel-HOWTO
  • Once your kernel is ready, type make dep, (you can also clean it with make
    mrproper and make clean) then type make bzfile (remember to put an empty
    floppy in your floppy drive so that you can boot from the floppy and see how
    good your kernel is. Remember to test all before you boot this kernel from
    the HD. Actually you can also do a make bzlilo. (it is the easiest way to
    run your SNA kernel) But remember that bzlilo will use the kernel even if it
    is the wrong one. Only use bzlilo at home!!!
  • Now that the kernel is done, our next step is to configure the
    etc-sna-0.00X.tar.gz so tar -zxvf etc-sna-0.00X.tar.gz and then go to the
    folder and type make.
  • Next step, gzip -zxvf asuite-0.0X.tar.gz and then (in the new folder) typemake.
  • sna-tools-0.0XX.tar.gz is the next one. so tar -zxvf sna-tools-0.0XX.tar.gz
    and the same with libsna-0.00X.tar.gz then copy the files lar.h and nof.h
    from libsna-0.00X folder to the sna-tools.0.0XX/include. In libsna-0.00X you
    will have to type ./configure and then type make. The same withsna-tools.0.0XX
